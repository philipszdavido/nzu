# nzu
A chainable terminal coloring Nodejs module (nzu is the Igbo language translate of chalk)

# Technologies
1. Nodejs

# Usage

* nzu.blue
* nzu.red
* nzu.cyan
* nzu.green
* nzu.yellow
* nzu.magenta
* nzu.white
* nzu.gray
